## P1-LightGame
### JavaScript

Esta práctica está hecha con HTML, CSS y JavaScript.

Es la recreación del juego de las luces, este se basa en un panel donde tienes que conseguir encender todas las luces. Al completarlo te sale un mensaje de enhorabuena, felicitándote por completar el juego.

Para la realización de este proyecto realicé una estructura y lógica inicial antes de la implementación del código, evitando así posible inconvenientes del desarrollo del proyecto.
